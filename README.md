# 桌面小猫原型

这是一个桌面宠物小游戏的静态原型，用来演示“桌面上有一只小猫在跑跳”的体验。当前版本使用 HTML/CSS/JS 模拟桌面、任务栏和文档场景。

## 体验方式

1. 启动一个静态服务器，例如：

   ```bash
   python3 -m http.server 8000
   ```

2. 访问 `http://localhost:8000`。

## 操作说明

- 小猫会在任务栏和文档之间自动跳跃。
- 点击桌面任意位置，小猫会朝点击方向跳过去。
